# 전북 현대 모터스 팀 선수 스타일 분석 및 K리그 랭킹

## 개요

이 문서는 전북 현대 모터스 팀의 모든 선수들의 스타일(롤)을 분석하고,
각 스타일별 K리그 전체 선수 랭킹에서의 위치를 보여줍니다.

**분석 기준:**

1. **롤 템플릿**: 데이터 기반 롤 템플릿 (풋볼 매니저 롤 정의 기반)

2. **적합도 점수 계산**:
   - 코사인 유사도 (60%): 방향 유사성 측정
   - 유클리드 거리 점수 (40%): 크기 차이 측정
   - 두 점수를 가중 평균하여 최종 적합도 산출 (0~100점)
   - 이 방식으로 점수 분포가 넓어져 선수 간 구분력 향상

3. **포지션별 랭킹**:
   - 같은 포지션 내에서만 롤 비교
   - 예: CM 포지션의 Deep Lying Playmaker는 CM 포지션 선수들과만 비교
   - 랭킹 근거: 포지션과 롤이 모두 동일한 선수들 간 비교

4. **표본 크기 보정 및 경기 수 보너스**:
   - 최소 기준: 5경기 이상, 200개 이벤트 이상
   - 베이지안 평균 방식으로 신뢰도 가중치 적용
   - 신뢰도가 낮은 선수(적은 경기/이벤트)는 평균 점수(50점)에 가까워지도록 보정
   - **경기 수 보너스**: 한 시즌 꾸준히 뛴 선수에게 가치 부여
     - 15경기 이상: +0.5점
     - 20경기 이상: +1.0점
     - 25경기 이상: +2.0점
     - 30경기 이상: +3.0점

5. **개선 방안 제안**:
   - 선수의 약점 지표 식별 (롤의 핵심 지표만 고려)
   - 상위 10명 선수와 비교하여 롤 내에서의 약점 식별
   - 구체적인 목표 수치 제시 (현재값, 목표값, 개선 필요량)
   - 검증 방법: 다음 시즌/경기 데이터에서 해당 지표 개선 여부 확인

**랭킹 검증 사례:**
- 정태욱 (CB, Ball Playing Defender): 14경기, 팀 승률 28.6% → 47위 (팀 승률 페널티 -1.0점 적용)
- 아론 (CB, 대전): 19경기, 팀 승률 15.8% → 낮은 순위 (팀 승률 페널티 -1.0점 적용)
- 경기 수 보너스와 팀 승률 기여도가 반영되어 실제 성과와 일치하는 랭킹 생성
